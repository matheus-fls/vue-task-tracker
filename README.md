# vue-task-tracker

## About

Basic task tracker built with Vue3.

Based on the great [crashcourse by TraversyMedia](https://www.youtube.com/watch?v=qZXt1Aom3Cs)

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
